Gramática del Nivel A1

1. Estructura de las preguntas:

W-Frage (+ complementos de tiempo, lugar o cantidad) + verbo + sujeto (+ complementos de tiempo, lugar o cantidad)

2. Estructura de las respuestas:

sujeto + verbo (+ complementos de tiempo, lugar o cantidad)

La W-Frage puede ser:


1. Wie? (¿Cómo?) => /vi?/

2. Wer? (¿Quién?) => /vea?/

3. Wo? (¿Dónde?) => /vo?/

4. Woher? (¿De dónde?) => /vojea?/

5. Welche? (¿Cuál?) => /velse?/

6. Wie viel? (¿Cuánto?) => /vi fil?/

7. Was? (¿Qué?) => /vas?/

8. Wann? (¿Cuándo?) => /van?/

9. Warum? (¿Por qué?) => /vagum?/

10. Wohin? (¿A dónde?) => /vojin?/

El verbo puede ser:


1. Infinitivo

2. Conjugado en presente

El sujeto puede ser:


1. ich (yo)

2. du (tú)

3. er (él)

4. sie (ella)

5. es (esto/ello)

6. wir (nosotros/as)

7. ihr (vosotros/as)

8. sie (ellos/as)

9. Sie (usted)

6. Países e idiomas:

"aus" (en alemán) significa "de" (en español) y se utiliza para países

"in" (en alemán) significa "en" (en español) y se utiliza para ciudades/municipios/pueblos/lugares

Ejemplos:

Olivia Miller kommt aus den USA - Olivia Miller viene de EE.UU

Gabriel Santos wohnt in San Francisco - Gabriel Santos vive en San Francisco

Boris Walder spricht Englisch und Deutsch - Boris Walder habla inglés y alemán

Saki Tanaka lernt Spanisch - Saki Tanaka estudia/aprende español
